# Whistle-Up — Find Your Perfect Coach

> Discover, compare, and connect with sports and fitness coaches near you — built for athletes, beginners, and seniors.

---

## What is Whistle-Up?

Whistle-Up is a sports coaching marketplace that helps users discover and connect with the right coach based on their needs.

The platform brings together athletes, beginners, and seniors with experienced coaches across multiple sports. Users can explore profiles, compare expertise, and make informed decisions — all through a smooth, modern interface.

Built as an MVP, Whistle-Up focuses on a frontend-first experience with realistic mock data and essential backend support for authentication and profile management.

---

## Features

- Browse coaches across sports and cities  
- Advanced filtering (sport, city, session type, age group)  
- Detailed coach profiles (experience, pricing, ratings, reviews)  
- Role-based authentication (Athlete / Coach)  
- Coach dashboard with editable profile  
- Athlete profile creation  
- Save favourite coaches (UI state)  
- Smooth animations and page transitions  
- Fully responsive design  

---

## User Roles

### Athlete
- Explore and filter coaches  
- View detailed profiles  
- Create personal profile  
- Save favourite coaches  
- View sessions (mock data)  

### Coach
- Create and manage public profile  
- Set pricing, sport, experience, session type  
- View public profile  
- Dashboard with stats and bookings (mock data)

## Installation & Setup

### 1. Clone the repository
git clone https://github.com/your-username/whistle-up.git  
cd whistle-up  

### 2. Install dependencies
npm install  

### 3. Setup environment variables
MONGO_URI=your_mongodb_connection_string  
JWT_SECRET=your_secret_key  
PORT=5000  

### 4. Run the app
npm run dev
